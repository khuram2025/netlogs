What happens when you apply an update

  1. Services stop (zentryc-web, zentryc-syslog) — web UI goes briefly offline.
  2. Current code tree is snapshot-backed-up; .env, venv, db.sqlite3, logs, backups are preserved.
  3. New code is unpacked, Python dependencies installed, OS packages updated if the release requires them.
  4. Services restart; a health-check confirms the new build is up.
  5. On any failure the agent automatically rolls back to the previous snapshot and restarts the old version.
